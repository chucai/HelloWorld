# React-native HelloWorld Project

## 介绍
包括三个页面: 搜索页面，搜索列表，产品页面。

## 总结
这个小程序虽然很小，但是包括的内容还是很多的。

1. 有页面直接的跳转，传递参数；
2. 列表展示信息；
3. 展示产品图片信息；
4. 导航栏；
5. 如何优雅的组织代码

在这次练习中，有如下几个方面没有涉及到：

1. 如何调试 react-native 代码
2. 修改等待加载页面
3. 增加一些动态效果
4. 与服务器端程序的交互: 保存数据
5. 应该写一个自动生成代码的工具: 生成 style 和  component 
6. 如何打包程序App
7. 没有使用开放的插件
